PHYS S-12: Introduction to Digital Fabrication

Faraz Bhuiyan, Harvard Summer School 2021

Assignments


01: Introduction & Documentation

Assignment 01: Create a website and sketch up an idea for a possible final project.

02: 2D Design & Cutting

Assignment 02: Make and document a circuit that includes an analog sensor. Use Arduino IDE to monitor the sensor value.

03: Electronics and Tools

Assignment 03: Make a circuit on a breadboard with components in the kit.

04: Rapid Prototyping

Assignment 04: Make and document a kinetic sculpture.

05: Programmable Electronics

Assignment 05: Program an Arduino board to make an output based on an input.

06: 3D Design & Printing

Assignment 06: Design and print an object. Scan something using a photogrammetry app.

07: Electronic Input Devices

Assignment 07: Make a sensor to measure a physical quantity with the Arduino.

08: Electronic Output Devices

Assignment 08: Program an Arduino to operate an output device.

09: Networking / IoT

Assignment 09: Demonstrate communication between microcontrollers.

10: Machine building

Assignment 10: Design a 3D mold, machine it, and cast material in it.

11: Computer Programming

Assignment 11: Make and document a kinetic sculpture.

12: Wildcard / Project Prep

Assignment 12: Document progress on a final project.
